## 🛠️ Instalación

1.  Instala una extensión de gestión de Scripts en tu navegador:
    *   [Tampermonkey](https://www.tampermonkey.net/) (Recomendado)
    *   [Violentmonkey](https://violentmonkey.github.io/)
2.  Haz clic en el archivo [**instagram-downloader.user.js**](https://github.com/jferariass/Instagram-Download-Pro/raw/main/instagram-downloader.user.js) en este repositorio.
3.  Tu extensión detectará automáticamente el script. Haz clic en **Instalar**.
4.  ¡Listo! Abre Instagram y verás el nuevo botón de descarga junto al icono de "Guardar".

---

## 🚀 Cómo funciona

El script utiliza un **Observador de Mutaciones (MutationObserver)** para inyectar botones dinámicamente cada vez que aparece una nueva publicación en tu Feed, Historia o sección de Reels. 

Cuando haces clic en el botón de descarga, el **Motor de Extracción Ultra (v2.1)** realiza un escaneo multinivel en el código fuente de la página para encontrar la URL real del archivo `.mp4` o `.jpg` alojado en los servidores de Facebook, permitiendo la descarga directa incluso cuando el vídeo está protegido.
